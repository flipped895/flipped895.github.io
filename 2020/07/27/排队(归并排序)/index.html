排队（归并排序）

排队

题面

牛客编程巅峰赛S1第5场 - 青铜&白银 C题
牛妹在银行排队等号时，观察到以下场景。
银行有m个服务窗口，假设当前有n个人等待办理业务，那么这n个人会被顺序分配一个从1到n的号码。
等待办理业务的流程如下：
从第1号到第n号顺序的进行排队。
假设当前第1号到第i-1号都正在办理或已经办理完业务，且某个窗口A没有客人正在办理业务，那么第i号会马上到窗口A办理他的业务。
如果有多个这样的窗口，第i号会随意选择一个窗口。
在0时刻，牛妹观察到m个窗口都没有客人正在办理业务，而n个人正在等待办理业务。
为了简化问题，我们假设第i号不管在哪个窗口办理业务，办理业务的时间都为ai。牛妹想知道有多少对(i,j)，满足1 ≤ i < j ≤n,且第i号办理业务完成的时间严格大于第j号办理业务完成的时间。对于30%数据，1≤n,m≤1000,对于100%数据，1≤n,m≤100000，1≤ai≤1e9。

题解（求逆序对）

简化题目,抽象建模,先求出每一个人最早完成时间(优先队列),求逆序对(树状数组离散化/归并排序/线段树+优先队列)


class Solution {
public:
    /**
     * 求解合法的(i,j)对的数量
     * @param n int整型 n个人
     * @param m int整型 m个窗口
     * @param a int整型vector 长度为n的vector,顺序表示1-n号客人的办理业务所需时间
     * @return long长整型
     */
    long long t[100007];
    long long b[100007];
    long long ans;
    long long getNumValidPairs(int n, int m, vector<int>& a) {
        priority_queue <long long,vector<long long>,greater<long long> > que;
        ans=0;
        for(int i=0;i<min(n,m);i++){//把最开始就能直接去窗口的人的完成时间保存
            t[i] = a[i];
            que.push(a[i]);
        }
        for(int i=m;i<n;i++){//求剩下的排队中的人的完成时间
            t[i] = (long long)a[i]+que.top();
            que.pop();//把最早完成的人出队,如果不出队始终都是同一个人在队首
            que.push(t[i]);
        }
        merge_sort(0,n-1);
        return ans;
    }
    void merge(int l,int mid,int r)//将两段有序数列按从小到大进行整合
    {
        int k=l;
        int p = l,q = mid+1;
        while(k<=r){
            if(p<=mid&&(q>r||t[p]<=t[q])){
                b[k++] = t[p];
                p++;
            }
            else{
                b[k++] = t[q];
                q++;
                ans += (mid-p+1);
            }
        }
        for(int i=l;i<=r;i++)//把已经整合好的两段序列覆盖到原序列中,没有这一步就不能保证已经走过的区间的有序性
            t[i] = b[i];
    }
    void merge_sort(int l,int r)//归并排序
    {
        if(l>=r)
            return ;
        int mid = (l+r)>>1;
        merge_sort(l,mid);
        merge_sort(mid+1,r);
        merge(l,mid,r);
    }
};
